## Sobre mim

Sou Engenheiro de Software com mais de seis anos de experiência em desenvolvimento backend com foco em Java e Spring Framework. Também possuo experiência com frontend utilizando React, Angular e Next.js.

Atualmente atuo no Grupo Boticário, no time de S&OP (Gestão de Demanda), contribuindo para soluções modernas e de alta disponibilidade.

Paralelamente, sou sócio e co-fundador da Vevox Soluções Digitais, onde desenvolvo o Fofito, um sistema completo de agendamentos multitenant com autenticação OAuth2, microsserviços Kotlin/Java, Next.js e uma arquitetura moderna orientada a produtos SaaS.

Minha atuação é guiada por arquitetura limpa, boas práticas, construção de sistemas escaláveis e colaboração multidisciplinar.

## Projetos em Destaque

### Vevox Soluções Digitais  
Empresa focada no desenvolvimento de soluções SaaS modernas, com arquitetura modular, multitenant e integração contínua.

### Fofito – Plataforma de Agendamentos  
SaaS completo para pet shops e clínicas veterinárias, com:
- Microsserviços em Java 21 com Spring Boot  
- Next.js 15 no frontend  
- Autenticação OAuth2 via Authorization Server próprio  
- Multi-tenant por schema  
- Integração com WhatsApp Business API  
- Infraestrutura em Kubernetes (k3s) com Traefik  

### Outros Projetos Relevantes
- Pipelines CI/CD para microserviços  
- Sistemas de gestão interna  
- Aplicações React e Angular para ERPs  
- APIs distribuídas e orientadas a eventos
